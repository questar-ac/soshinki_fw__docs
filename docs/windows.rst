==================
Running on Windows
==================

| Windows 10または11を使っている場合は、 `Windows Subsystem for Linux (WSL) <https://en.wikipedia.org/wiki/Windows_Subsystem_for_Linux>`_ を利用してsoshinki_fwを実行することができます。
| You can run soshinki_fw on Windows 10 or 11 by using `Windows Subsystem for Linux (WSL) <https://en.wikipedia.org/wiki/Windows_Subsystem_for_Linux>`_.
